🚀 Release Notes – June 2025 – Week 1

Explore the latest updates across Email Whitelisting, VoIP Calling, Location Filtering, and Analytics Enhancements.

📨 Email Whitelisting for Ticket Forwarding

Control where tickets can be sent or forwarded using a centralized whitelist.

🔐 Admin Controls:

  • Manage domains and individual emails via intuitive UI.
  • Enable or disable the whitelist globally at any time.

✉️ Agent Experience:

  • Real-time validation ensures forwarding is restricted to approved addresses.
  • Enhances control and reduces data leakage risks.

📞 VoIP Calling (Powered by Exotel)

Let agents make and receive browser-based calls from within the platform.

📤 Outbound (Click-to-Call):

  • Click-to-call buttons beside phone numbers and author profiles.

📥 Inbound Call Handling:

  • Answer support number calls and auto-create tickets for new contacts.

📝 Post-Call Experience:

  • Agents enter notes during wrap-up.
  • Call recordings appear in the ticket’s mention card.

🎛 IVR and CSAT Integration:

  • Caller IVR input logged as ticket attributes.
  • Optional post-call CSAT survey stored in CSAT/NPS form or custom field.

📍 Location Profile Filtering

Apply location profile filters across major modules like Inbox, Reports, and Workflows.

🗂 Inbox Filtering:

  • Filter mentions/tickets by state, city, name, or location tags.

📊 Report Generation:

  • Export Excel reports scoped by location attributes.

⚙️ Workflow Automation:

  • Use location criteria to trigger actions in workflow rules.

📈 Analytics Enhancements

Make data-driven decisions with new filters, toggles, and benchmarking tools.

🆔 Mention ID on Hover:

  • Hover to reveal Mention ID and copy it for search.

🔁 FLR Toggle:

  • Created Date: View tickets created in timeframe.
  • Modified Date: View tickets updated in timeframe.

⚖️ Fair Benchmarking (Public Engagement Only):

  • New checkbox filters benchmarking by public metrics only.
  • Included Metrics:
    • Twitter/X: Likes, Replies, Retweets, Quote Tweets
    • Instagram: Likes, Comments
    • Facebook: Reactions (Like, Love, Wow...), Comments
    • YouTube: Views, Likes, Comments
